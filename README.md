
# Telegram-бот для безопасных платежей с Daytona

Этот Telegram-бот использует Daytona для создания временных, изолированных окружений с браузером Firefox в полноэкранном режиме (kiosk) для безопасного проведения онлайн-платежей.

## Функционал

- **Создание сессий:** Быстрое создание sandbox-окружений для Spotify, Discord или любой другой ссылки.
- **VNC-доступ:** Получение прямой ссылки на VNC-сессию в браузере для управления окружением.
- **Kiosk-режим:** Firefox запускается в полноэкранном режиме без лишних элементов интерфейса для максимальной безопасности.
- **Автоматическое удаление:** Каждая сессия автоматически удаляется через 7 минут.
- **Прокси-менеджер:** Базовый функционал для добавления и использования прокси (в разработке).

## 1. Предварительные требования

- **Аккаунт Daytona:** Зарегистрируйтесь на [платформе Daytona](https://www.daytona.io/).
- **Токен Daytona API:** Создайте и скопируйте ваш API-ключ из [панели управления Daytona](https://app.daytona.io/settings/api-keys).
- **Токен Telegram-бота:** Создайте нового бота в Telegram через [@BotFather](https://t.me/BotFather) и получите его токен.
- **Python 3.10+**
- **Git**

## 2. Настройка Daytona: Создание снапшота

Снапшот — это шаблон для ваших sandbox-окружений. Его нужно создать один раз.

1.  **Установите Daytona CLI:**

    ```bash
    brew install daytonaio/cli/daytona
    ```

2.  **Войдите в свой аккаунт:**

    ```bash
    daytona login
    ```

3.  **Создайте снапшот из репозитория:**
    Daytona CLI автоматически использует Dockerfile из репозитория для сборки образа.

    ```bash
    daytona snapshot create firefox-vnc-snapshot --dockerfile ./Dockerfile
    ```

    Дождитесь завершения сборки. После этого снапшот с именем `firefox-vnc-snapshot` будет готов к использованию.

## 3. Настройка и запуск бота

1.  **Клонируйте репозиторий:**

    ```bash
    git clone https://github.com/loxeblan0-sys/Mmm.git
    cd Mmm
    ```

2.  **Установите зависимости:**

    ```bash
    pip install pytelegrambotapi daytona-sdk
    ```

3.  **Настройте переменные окружения:**
    Создайте файл `.env` в корневой директории проекта и добавьте в него ваши токены:

    ```
    TELEGRAM_BOT_TOKEN="ВАШ_ТЕЛЕГРАМ_ТОКЕН"
    DAYTONA_API_KEY="ВАШ_DAYTONA_API_КЛЮЧ"
    ```

    Или экспортируйте их в консоли:

    ```bash
    export TELEGRAM_BOT_TOKEN="ВАШ_ТЕЛЕГРАМ_ТОКЕН"
    export DAYTONA_API_KEY="ВАШ_DAYTONA_API_КЛЮЧ"
    ```

4.  **Запустите бота:**

    ```bash
    python3 bot.py
    ```

Бот запущен и готов к работе. Откройте чат с вашим ботом в Telegram и используйте кнопки для создания сессий.

## Структура проекта

- `bot.py`: Основной код Telegram-бота.
- `Dockerfile`: Файл для создания Docker-образа с Firefox, VNC и необходимыми утилитами.
- `start.sh`: Скрипт, который запускается внутри контейнера для старта VNC и Firefox.
- `proxies.json`: Файл для хранения списка прокси (создается автоматически).
- `README.md`: Эта инструкция.
