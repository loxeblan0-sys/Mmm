# Telegram-бот для безопасных платежей с Daytona

Этот Telegram-бот использует Daytona для создания временных, изолированных окружений с браузером Firefox в полноэкранном режиме (kiosk) для безопасного проведения онлайн-платежей.

## Финальная инструкция по запуску (с нуля)

Выполни эти шаги в точности, и бот заработает.

### Шаг 1: Подготовка

1.  **Аккаунт Daytona**: Убедись, что у тебя есть аккаунт на [Daytona](https://www.daytona.io/).
2.  **Daytona API Key**: Зайди в [панель управления Daytona](https://app.daytona.io/settings/api-keys) и создай API-ключ. **Скопируй и сохрани его**, он понадобится.
3.  **Telegram Bot Token**: Создай нового бота в Telegram через [@BotFather](https://t.me/BotFather), получи его токен и тоже сохрани.
4.  **Установи Git и Python**: Если их нет, установи `git` и `python` (версии 3.10 или новее).

### Шаг 2: Настройка Daytona (создание снапшота)

Это **самый важный шаг**. Его нужно сделать один раз. Мы создадим шаблон (снапшот) для будущих сессий бота.

1.  **Установи Daytona CLI** (командную строку Daytona):

    ```bash
    brew install daytonaio/cli/daytona
    ```
    *(Если у тебя не macOS, смотри [другие способы установки](https://www.daytona.io/docs/en/getting-started#cli)).*

2.  **Авторизуйся** в Daytona через CLI:

    ```bash
    daytona login
    ```
    Эта команда попросит вставить твой API-ключ из Шага 1.

3.  **Клонируй репозиторий** с кодом бота:

    ```bash
    git clone https://github.com/loxeblan0-sys/Mmm.git
    cd Mmm
    ```

4.  **Создай снапшот** из `Dockerfile` в репозитории. Имя снапшота должно быть в точности `ubuntu-4vcpu-8ram-100gb`, так как оно зашито в код бота.

    ```bash
    daytona snapshot create ubuntu-4vcpu-8ram-100gb --dockerfile ./Dockerfile
    ```

    Дождись, пока команда выполнится. Daytona соберёт Docker-образ и зарегистрирует его как снапшот. В конце ты увидишь сообщение об успешном создании. Проверить статус можно в [панели управления Daytona](https://app.daytona.io/snapshots).

### Шаг 3: Запуск Telegram-бота

Теперь, когда снапшот готов, можно запускать самого бота.

1.  **Установи зависимости** для Python:

    ```bash
    # Убедись, что ты находишься в папке Mmm
    pip install pytelegrambotapi daytona-sdk
    ```

2.  **Настрой переменные окружения**. Это секретные ключи, которые нельзя хранить в коде. Укажи их в терминале перед запуском.

    ```bash
    export TELEGRAM_BOT_TOKEN="СЮДА_ВСТАВЬ_ТОКЕН_ТВОЕГО_БОТА"
    export DAYTONA_API_KEY="СЮДА_ВСТАВЬ_API_КЛЮЧ_DAYTONA"
    ```

3.  **Запусти бота!**

    ```bash
    python3 bot.py
    ```

Если всё сделано правильно, в консоли появится сообщение `Bot is running...`. Теперь можешь зайти в Telegram, найти своего бота и пользоваться им.

---

## Как это работает

-   **`Dockerfile`**: Описывает, как собрать образ на базе Debian с установленным Firefox, VNC-сервером и рабочим окружением XFCE.
-   **`start.sh`**: Скрипт, который запускается при старте каждого контейнера. Он поднимает виртуальный экран (Xvfb), VNC-сервер (x11vnc), веб-клиент VNC (noVNC) и запускает Firefox в kiosk-режиме с нужным URL.
-   **`bot.py`**: Код бота, который слушает команды в Telegram, обращается к Daytona API для создания `sandbox` из твоего снапшота и выдаёт временную ссылку на VNC.
